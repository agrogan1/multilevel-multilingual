4.1 Two Level Model

An unconditional multilevel model is a model with no independent variables. One should always run an unconditional model as the first step of a multilevel model in order to get a sense of the way that variation is apportioned in the model across the different levels.

4.1.1 The Equation

\[\text{outcome}_{ij}= \beta_0 + u_{0j} + e_{ij} \tag{4.1}\]

The Intraclass Correlation Coefficient (ICC) is given by:

\[\text{ICC} = \frac{var(u_{0j})}{var(u_{0j}) + var(e_{ij})} \tag{4.2}\]

In a two level multilevel model, the ICC provides a measure of the proportion of variation attributable to Level 2.

4.2 Three Level Model

4.2.1 The Equation

\[\text{outcome}_{ij}= \beta_0 + u_{0j} + v_{0i} + e_{ij} \tag{4.3}\]

As discussed in the main text, in a three level model, there are two intraclass correlation coefficients (StataCorp, 2023). The formulas for the Intraclass Correlation Coefficient (ICC) are given by (StataCorp, 2023):

\[\text{ICC} = \frac{var(u_{0j})}{var(u_{0j}) + var(v_{0i}) + var(e_{ij})} \tag{4.4}\]

Again, following StataCorp (2023), Equation eq-ICCunconditional3A is the correlation of responses for person-timepoints from the same country but different persons.

\[\text{ICC} = \frac{var(u_{0j}) + var(v_{0i})}{var(u_{0j}) + var(v_{0i}) + var(e_{ij})} \tag{4.5}\]

Again, closely following StataCorp (2023), Equation eq-ICCunconditional3B is the correlation of responses for person-timepoints from the same country and same person.
